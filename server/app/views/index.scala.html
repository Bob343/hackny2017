Locate

We use the Foursquare API to locate any places within 100 meters of your location. Then we have a list of venues you might be looking at. For each of the possible venues we retrieve a set of photographs from Foursquare.

code

Train

We create a custom Clarifai model and train it with the pictures of nearby venues. Each set of pictures is assigned a label of the venue name so we can identify similar pictures later.

label

Predict

Take a picture submit it to the server. The server then sends The picture to the specially trained Clarifai model. Clarifai then predicts the best location match based on the training data provided.
